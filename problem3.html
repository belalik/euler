Largest Prime Factor

The prime factors of 13195 are 5, 7, 13 and 29.

What is the largest prime factor of the number 600851475143 ?

Sample Code: 


/* My original implementation was right 
** but too inefficient.  It was basically a 
** brute force approach, based on a isPrime() function. 
** It worked on JFiddle though, so I got the answer.   
** I then read through Project Euler's overview 
** and decided to apply their algorithm and move on. 
** This is it:  
*/

if (num % 2 === 0) {
	num = num/2;
	lastFactor = 2;
	while (num % 2 === 0) {
		num = num/2;
	}
}
else {
	lastFactor = 1;
}
	
// This variable holds the answer
factor=3;

while (num>1) {
	if (num % factor===0) {
		num = num/factor;
		lastFactor = factor;

		while (num % factor===0) {
			num = num/factor;
		}
	}
	factor = factor+2;
}
