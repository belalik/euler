The Odin Project

Some time ago I decided to check out the Odin Project - a free online web development course.

This page is part of the Odin course - using Javascript to solve some simple Project Euler problems.

Rather than making a very simple text page, I looked around for free templates that would give it a style and allow me to play a bit more with the front-end. I stumbled upon this one which looked good enough foundation for what I wanted.

Open Source Web Design is a great site which I highly recommend for nice simple designs. It's also a good resource for studying applied HTML/CSS in 'practice'.

This page is overly complicated for its purpose. It could be done with a simpler, cleaner layout. But please remember the intention here is to play around with menus, logos, colours etc., and add a bit of unnecessary text to test things out ;-)
